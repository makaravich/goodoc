Plugin to import Google Documents into a WordPress site


== Description ==

Plugin to import Google Documents into a WordPress site.

Struggling to copy and paste your Google Docs content into WordPress while maintaining perfect formatting? This plugin
is your one-stop solution! Seamlessly import your Google Docs directly into your WordPress site, complete with all the
formatting, images, and styles you've meticulously crafted.


== Frequently Asked Questions ==

= What does this plugin do? =

This plugin allows you to import your Google Docs directly into WordPress posts, saving you time and effort.

= Why do I need to register on the server? =

Registration on our server is necessary to ensure interaction between you, as a Google Documents user, and the plugin.
All complex procedures related to registration and configuration of the Google application for interaction with the
Google Documents API have already been resolved on our server. And you don't need to go through them yourself. All you
need to do is register, get a token, and enter it on the plugin settings page.

= Is this plugin free? =

Yep! The plugin is fully free. No credit card is needed.

= Does the plugin import images from Google Docs? =

Yes, the plugin can import images directly from Google documents into the media library of your WordPress site.
Additional alignment of images may be required after import.

= Is it safe to connect my Google account to the plugin? =

The plugin uses secure authorization protocols to connect to your Google account. We do not store or transmit your data
in plain text. They are all encrypted.

= Is the GooDoc plugin compatible with my WordPress theme, or with my other plugins? =

Probably yes. Although we are unable to test the plugin for compatibility with any existing themes and plugins, we can
say that the plugin is written using modern coding standards. We strived to make it as user-friendly and code-friendly
as possible.

= I'm having trouble connecting my Google account. What should I do? =

Read our [post](https://doc-r.com/quick-start/) about connecting the plugin to your Google account. There are
step-by-step instructions there. If you still have questions, please contact us using the feedback form.


== Installation ==
1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `WP GooDoc` and click 'Install Now' or click on the `upload` link to upload `wp-goodoc.zip`
4. Click on `Activate plugin`
5. Create an account on our [website](https://doc-r.com/login/). After completing the registration, an email will be
sent containing an access token to the service.
6. Go to the Dashboard of your WordPress site and select GooDoc Options in the GooDoc Import menu item. Then, on the
page that opens, in the GooDoc Client Token field, copy and paste the token you received earlier in the email. Save it
by clicking the Save button.
7. Now you can go to the import page by selecting the GooDoc Import menu item, select the desired document and import it
to your WordPress site by clicking the Start Import button.
